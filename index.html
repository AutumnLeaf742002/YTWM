Tecnologias que utilizamos

React

Arquitectura basada en componentes, rendimiento mejorado con el DOM virtual y un ecosistema maduro.

React Native

Desarrollo nativo con JavaScript, reutilización de código y rápida iteración en múltiples plataformas.

Electron

Permite crear aplicaciones de escritorio multiplataforma usando tecnologías web, con un solo código base.

TailwindCSS

Diseño altamente personalizable, desarrollo más rápido con utilidades predefinidas y consistencia en el estilo.
